# 3D Print Shop Website

This is a redesigned 3D Print Shop website with:
- Cart system and PayPal checkout
- Product listing from products.json
- Mystery images above products
- Click and success sounds

## How to update

1. **Logo:** Replace `images/logo.png`.
2. **Product images:** Replace files in `images/` and update `products.json` for name, price, description, and mysteryImage.
3. **Sounds:** Replace `sounds/click.mp3` or `sounds/success.mp3`.
